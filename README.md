## О проекте

Приложение построено на архитектуре **Flask Blueprints**: каждая лабораторная работа — отдельный модуль с собственными маршрутами и шаблонами. Общие компоненты (база данных, модели, права доступа, логирование посещений) вынесены в корневые модули каталога `app/`.

Главная страница (`/`) содержит навигацию ко всем шести лабораторным работам.

---

## Технологии

| Компонент | Библиотека |
|-----------|------------|
| Веб-фреймворк | Flask 3.0 |
| ORM | Flask-SQLAlchemy |
| Аутентификация | Flask-Login |
| Миграции | Flask-Migrate + Alembic |
| Генерация тестовых данных | Faker |
| Тестирование | pytest, pytest-mock |
| WSGI-сервер (production) | Gunicorn |

База данных — **SQLite** (`app/lab4.db`).

## Быстрый старт

### 1. Клонирование и установка зависимостей

```bash
git clone <url-репозитория>
cd web_labs_gubin_241_371

python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
cd app
python3 app.py
```

Приложение будет доступно по адресу [http://127.0.0.1:5000](http://127.0.0.1:5000).

Альтернативный способ через Flask CLI (из каталога `app/`):

```bash
export FLASK_APP=app
export FLASK_DEBUG=1
flask run
```

### 3. Production-запуск (Gunicorn)

```bash
cd app
gunicorn -w 4 -b 0.0.0.0:8000 'app:application'
```

### Лабораторная №4 — База данных и CRUD пользователей

**URL:** `/lab4/`

Полноценная система управления пользователями на SQLite с ролями, хешированием паролей и разграничением прав.

| Маршрут | Описание |
|---------|----------|
| `/lab4/` | Список пользователей |

## Миграции базы данных

Миграции управляются через Flask-Migrate. Команды выполняются из каталога `app/`:

```bash
# Применить все миграции
flask db upgrade

# Создать новую миграцию после изменения models.py
flask db migrate -m "описание изменений"

# Откатить последнюю миграцию
flask db downgrade
```

Файл базы данных создаётся автоматически при первом запуске (`app/lab4.db`). Таблицы для курсов и отзывов добавляются миграцией `a1b2c3d4e5f6_add_courses_and_reviews_tables`.
